# Best Hobby Event

베스트하비 고객이 이벤트, 예약, 마감 정보를 한곳에서 확인할 수 있도록 만든 모바일 우선 웹앱 MVP입니다.

## MVP 기능
- 전체 이벤트 허브 홈
- 브랜드별 필터
- 이벤트 상세 보기
- 마감 임박 / 진행 중 / 오늘 시작 상태 표시
- 관심 브랜드 저장

## 브랜드
- 굿스마일컴퍼니
- 코토부키야
- 메가하우스
- T-SPARK

## 실행 방법
```bash
npm install
npm run dev
```

브라우저에서 아래 주소로 접속합니다.

```bash
http://localhost:3000
```

## 빌드
```bash
npm run build
```
